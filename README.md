# Todo List

A todo list built with the MERN stack (Mongo, Express, React, Node).

### Install Server and Client Dependencies

```bash
$ npm install
$ npm run client-install
```

## Available Scripts

### Start server

```bash
$ npm run start
```

### Start server (in development mode)

```bash
$ npm run server
```

### Start client

```bash
$ npm run client
```

### Start client and server

```bash
$ npm run dev
```
